# Getting Started

### Reference Documentation

**`Terra`** 大地女神  意味着根本 基础。

这个项目用来写一些demo来理解 Java、设计模式等基础的的东西。

<ol>
    <li>proxy包: 使用自定义annotation完成aop注入。设计模式:代理模式、工厂模式</li>
    <li>io包: Nio 基于 channel buffer selector</li>
    <li>jmm包: Volatile</li>
    <li>pattern.design包: 设计模式包 Factory Singleton Proxy</li>
</ol>
